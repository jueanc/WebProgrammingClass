Website Overview 網站結構介紹

  • Home (Index.html) — 個人首頁，為雙欄版面：左側為個人簡介與頭像（點擊頭像可前往 About.html），右側為主內容。
  • About.html — 介紹學校、興趣與音樂。將滑鼠移到 Fire EX. / Darvish 圖片上時，會播放對應的背景影片。
  • Highlights.html — 以時間軸呈現重要事件，搭配技能與成就展示。滑鼠移到「Negogiyup」文字時，會觸發 Rickroll 背景影片。
  • Table.html — 包含可自訂樣式的成績表格、九九乘法表與座位表，並搭配互動按鈕操作。
  • Gallery.html — 包含 Canvas 城市日夜切換動畫、Polaroid 照片切換器與 Soto 圖片 Hover 特效。
  • Description.html — 專門用來說明本網站的整體架構、功能對應的 Lab，以及實作過程。

How to Use & Features 操作說明與功能

  • 導覽列 Navigation Bar — 點選上方各頁名稱，可在 Home / About / Highlights / Table / Gallery / Description 之間切換，目前所在頁面會加粗並有底線標示。
  • 頭像 Image Map — Home 頁面中，將滑鼠移到頭像臉部區域並點擊，可快速前往 About.html（關於頁）。
  • 背景影片互動 — 在 About 頁，將滑鼠移到 Darvish / Fire EX. 圖片上會啟用對應背景影片；在 Highlights 頁，將滑鼠移到「Negogiyup」連結上會出現 Rickroll 背景影片。
  • 表格控制 Table Styler — Table 頁可以透過按鈕調整表格寬度、邊框與間距、底色等樣式，並使用「Averages」計算各科平均成績，「Reset」可恢復原始樣式。
  • 九九乘法表 Multiplication Table — 點擊「Show Multiplication Table」按鈕會產生 1–9 的九九乘法表，再次點擊可收合隱藏。
  • 座位表 Seating Chart — 下方座位表在滑鼠移上去時會改變底色，用以標示不同座位區域與位置，方便辨識。
  • Canvas 日夜切換 — 在 Gallery 頁面按下「切換白天」按鈕，可在白天與夜晚的城市場景之間切換，包含天空顏色、星空與建築燈光的變化。
  • Polaroid 圖片切換 — 使用「Prev / Next」按鈕或下方彩色小方塊可以切換不同照片，下方文字會同步更新圖片標題與「Record x / 6」計數。
  • Soto Hover 特效 — 將滑鼠移到 Soto 圖片上，會切換成生氣版本並顯示「I said DON'T TOUCH ME」提示文字。
  • 頁面淡入效果 — 進入各頁時，主內容區會有淡入動畫，讓頁面切換過程更平滑、不突兀。

Lab Mapping（與課堂 Lab 的對應）

  • Lab 2 — 多頁導覽列、一致的主題風格、Home 雙欄排版（側邊個人資訊＋主內容）。
  • Lab 3 — Image Map（Home 頁頭像可點擊）、Hover Overlay（Gallery 頁 Soto 圖片 Hover 效果）。
  • Lab 4 — Table Styler（可調整寬度、邊距、色彩、計算平均值、Reset 功能）＋ 九九乘法表的動態產生。
  • Bonus — Canvas 畫布創作：城市日夜變化、星空與建築燈光效果。

JavaScript Files（外部 JS 檔案功能）

  • js/GoHome.js — 負責 Home 頁頭像的點擊事件：點擊後跳轉至 About 頁，並在點擊時顯示小彩蛋效果。
  • js/bgVideo.js — 控制 About 頁背景影片的播放與暫停，依照滑鼠是否在圖片上決定是否播放。
  • js/rickroll.js — 控制 Highlights 頁「Negogiyup」區塊的 hover 事件，觸發 Rickroll 背景影片。
  • js/table.js — 負責 Table 頁的表格樣式調整、平均值計算，以及九九乘法表的動態產生與顯示／隱藏。
  • js/gallery.js — 控制 Gallery 頁的 Canvas 日夜動畫切換，以及 Polaroid 圖片切換與計數顯示。

Requirement Checklist（作業需求對照）

  • 至少 5 頁以上之多頁網站，並透過導覽列在各頁之間切換。
  • 至少一頁為 2 欄版面設計（Home 頁：左側個人資訊＋右側主內容）。
  • 至少一個 Image Map（Home 頁頭像）與一個 Hover Overlay 效果（Gallery 頁 Soto 圖）。
  • JavaScript 以外部檔案形式載入，依功能拆分不同檔案管理。
  • 包含互動式表格功能、九九乘法表，以及額外的 Canvas 畫布加分項目。

實作過程 Process

  • 以舊版個人網頁為基礎，保留原本結構，再重新設計整體風格與版面。
  • 調整配色與字體，讓畫面更乾淨並維持主題一致性（側欄風格、背景影片、圖片濾鏡等）。
  • 將課堂作業成果（Canvas 城市、Polaroid 切換、Table Styler 等）整合進同一個主題網站中。
  • 原本分散在各頁的 JavaScript 整理為獨立外部檔案，方便維護與閱讀。
  • 新增 Description.html 作為「說明書」，讓老師與同學可以快速了解整個網站的結構與實作重點。
