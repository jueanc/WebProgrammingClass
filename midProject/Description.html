Website Overview 介紹

  • Home (Index.html) — 個人首頁、雙欄版面、左側圖像可跳轉至Index.html
  • About.html — 介紹學校、興趣與音樂，(FireEX. / Darvish)圖片鼠標移至上方會有背景影片
  • Highlights.html — 時間軸、技能與成就展示，鼠標移至Negogiyup會有背景影片
  • Table.html — 可自訂樣式表格、九九乘法表與座位表
  • Gallery.html — Canvas 城市日夜切換、Polaroid 切換器、Soto Hover 特效
  • Description.html — 專案說明與實作過程

How to Use & Features 操作說明

  • 導覽列 Navigation Bar — 點選上方各頁名稱可在 Home / About / Highlights / Table / Gallery / Description 間切換，當前頁面會有底線與加粗標記。
  • 頭像 Image Map — Home 頁面中，將滑鼠移到頭像臉部區域並點擊，可快速前往About.html（關於）。
  • 背景影片互動 — About 頁將滑鼠移到 Darvish / Fire EX. 圖片上，會啟用對應背景影片；Highlights 頁將滑鼠移到「Negogiyup」連結上，會出現 Rickroll 背景影片。
  • 表格控制 Table Styler — Table 頁可以透過按鈕調整表格寬度、邊框與間距、底色，並使用「Averages」計算各科平均，「Reset」恢復原始樣式。
  • 九九乘法表 Multiplication Table — 點擊「Show Multiplication Table」按鈕產生乘法表，再次點擊可收合。
  • 座位表 Seating Chart — 下方座位表支援滑鼠 hover，部分座位有不同底色標示講桌、走道或特別座位。
  • Canvas 日夜切換 — Gallery 頁面按下「切換白天」按鈕，可在白天與夜晚的城市場景之間切換，包含天空顏色、星空與建築燈光變化。
  • Polaroid 圖片切換 — 使用「Prev / Next」按鈕或下方彩色小方塊切換照片，下方會同步更新圖片標題與「Record x / 6」計數。
  • Soto Hover 特效 — 將滑鼠移到 Soto 圖片上，會切換成生氣版本並顯示「I said DON'T TOUCH ME」文字。
  • 頁面淡入效果 — 進入各頁時，主內容區會有淡入動畫，讓不同頁面切換更平滑。
