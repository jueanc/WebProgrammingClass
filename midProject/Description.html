Website Overview

  • Home (Index.html) — 個人首頁、雙欄版面、形象介紹、Profile點即可跳轉至About.html
  • About.html — 介紹學校、興趣與音樂，背景影片 (FireEX. / Darvish)
  • Highlights.html — 時間軸、技能與成就展示
  • Table.html — 可自訂樣式表格、九九乘法表與座位表
  • Gallery.html — Canvas 日夜場景、Polaroid 切換器、Soto Hover 特效
  • Description.html — 專案說明與實作過程

Lab Mapping

  • Lab 2 — 多頁導覽、一致主題、雙欄排版
  • Lab 3 — Image Map（頭像）、Hover Overlay（Soto 圖）
  • Lab 4 — Table Styler（寬度、邊距、色彩、平均值、Reset）+ 九九乘法表
  • Bonus — Canvas 畫布創作（日夜變化、星空與建築）

JavaScript Files

  • js/GoHome.js — 點擊頭像回首頁動畫
  • js/bgVideo.js — About 背景影片播放控制
  • js/rickroll.js — Highlights 背景影片 hover 播放
  • js/table.js — 表格控制、平均值、九九乘法表
  • js/gallery.js — Canvas 日夜動畫、Polaroid 圖片切換

Requirement Checklist

  • 5 頁以上、多頁導覽一致
  • 2 欄版面（左側個人資訊＋右側主內容）
  • Image Map（頭像）、Hover Overlay（Soto 圖）
  • 外部 JS 檔案結構化載入
  • 表格互動、九九乘法表、Canvas（Bonus）

Notes

  • 圖片路徑：../Photo/

實作過程

  • 從舊的個人網頁修改，保留整體結構但重新設計風格
  • 調整顏色與字體，使主題更簡潔有一致性
  • 整合課堂作業成果（Canvas、Polaroid、Table）
  • 原先寫在各頁內的 JS 改為獨立外部檔案以便維護
  • 新增 Description.html 用於專案總結與展示
