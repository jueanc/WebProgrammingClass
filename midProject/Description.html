Midterm Project Description

Dream Space Personal Website

Website Overview

  • Home (Index.html) — Personal intro + background video hover (Fire EX. / Darvish)
  • Highlights.html — Journey timeline, skills & awards
  • Resource.html — External links，RickRoll 背景影片 hover
  • Table.html — Table Styler + 2–9 Multiplication + Seating chart
  • Gallery.html — Canvas Day/Night + Polaroid switcher + Hover overlay

Lab Mapping

  • Lab 2 — Multi-page, consistent navbar, 2-column layout
  • Lab 3 — Image Map（左欄頭像）、Hover Overlay（Soto 圖）
  • Lab 4 — Table Styler（寬度 / 邊距 / 色彩 / Reset / 平均）、九九乘法表
  • Bonus — Canvas creative artwork（漸層天空、星星/雲、建築、標題文字）

JavaScript Files

  • js/GoHome.js — 點擊側邊頭像：閃眼→返回 Home
  • js/bgVideo.js — Home 背景影片 hover 播放
  • js/rickroll.js — Resource 背景影片 hover 播放
  • js/table.js — 表格樣式控制、平均值、2–9 乘法表
  • js/gallery.js — Canvas 日夜動畫 + Polaroid 圖片切換

Requirement Checklist

  • 5 頁以上、多頁一致導覽與主題
  • 2 欄版面（左側個人資訊＋右側主內容）
  • Image Map（頭像）、Hover Overlay（Soto 圖）
  • JS 全為外部檔案，頁面按需載入
  • Table Styler + Multiplication + Canvas（Bonus）
